CRUD App with PyQt5 (MySQL + CSV + Admin Login)

A desktop CRUD (Create, Read, Update, Delete) application built with PyQt5, using MySQL for main data storage and CSV for optional export.  
The app features an **Admin Login system** that authenticates users before granting access to the main CRUD interface.

This project is ideal for learning PyQt5 GUI development, database integration, file handling, and authentication in Python.


Features

- Admin Login Authentication
- Secure redirection to the CRUD dashboard after login
- Create new records
- Read and display records from MySQL
- Update existing records
- Delete records
- Export records to CSV
- Clean and user-friendly PyQt5 interface
- Input validation and error handling


Technologies Used

- Python 3  
- PyQt5  
- MySQL (via `mysql-connector-python` or `PyMySQL`)  
- CSV (export feature)
